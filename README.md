## EduCar - Online ixtisas seçimi

Hər şeyin sürətlə rəqəmsallaşdığı dövrdə abituriyentlər hələdə ["Abituriyent"](http://www.abiturient.az/) jurnalının çap edilməsini gözləyir. Halbuki, jurnallardakı bütün məlumatlardan vebsayt vasitəsilə daha interaktiv formada istifadə edə bilərik.

"EduCar" layihəsinin məqsədi abituriyent və vaildeynlərinin ölkədəki univeristetlər və ixtisaslar haqqındakı bütün sualları interaktiv şəkildə cavablandırmaqdır.

Data tarixçəmiz böyüdükcə ixtisas sahələri və universitetlərin gələcəkdəki potensialını da proqnozlaşdırmaq mümkün ola bilər. Hal-hazırda əlimizdə olan bütün məlumatları datasetimizə əlavə etməyə davam edirik.

### Layihəyə töhfə vermək üçün edə biləcəkləriniz

* **Kəşfiyyatçı analiz (Exploratory Data Analysis)** - Vizuallaşdırmaq üçün seçdiyimiz çartların yaxşılaşdırmağımıza kömək edə və ya yeni çartlar təklif edə bilərsiniz.

* **Verilənlər bazasının böyüdülməsi (Data Mining)** - Abituriyent jurnallarının köhnə buraxlışlarını əldə etməyizə və bu məlumatları verilənlər bazamıza əlavə etməklə layihəmizə dəstək ola bilərsiniz. 

* **İstifadəçi interfeysin inkişaf etdirilməsi (UI/UX Development)** - İstifadəçilərin vebsaytdan istifadəsini rahatlaşdırmaq üçün təkliflərinizi bizimlə bölüşə bilərsiniz.
